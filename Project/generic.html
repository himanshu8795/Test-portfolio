Project YreX

To identify need , is the most important part of software Analysis and so for us too .
To identify needs we should have known about the Project Title ,” Media Sharing & Chatting Application “ .Title tells us that it is an application for media sharing and chatting .
So the needs found to be are :
1. To develop a single application performing two tasks .
2. To develop an application which can perform Chatting and Media Sharing both simultaneously .
3. To develop an application which doesn't use Mobile Data or can say Internet .
4. To develop an application which is fast enough to save time .
5. To develop an application which can share any media.
6. To develop an application which uses less resources and should be lightweight.

Preliminary Investigation plays an important role in Software Analysis as to know the problems and deficiency of previous or existing applications makes it easier to develop a new application .
So the Investigation done are :
1. There are many chatting applications which work on the Internet or require Mobile Data to transfer messages .
2. There are many media sharing applications that also require the Internet and many of those don’t . For this , those applications first upload media to their servers and then send it to client or user .
3. Uploading and downloading wastes too much time .
4. Both chatting and media sharing in a single application is not seen yet .
5. Speed and accuracy is lacking in existing applications .

So , from the above reasons and statements we can get the need to develop our Application .
